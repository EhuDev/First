Services

  • Website Design

    Web design refers to the process of creating and styling the visual appearance of a website or web application. It involves designing the layout, color scheme, typography, and other visual elements that make up the look and feel of a website. Some common responsibilities of a web designer may include: Planning and creating the overall layout and structure of a website or web application Designing and creating user interface elements such as buttons, menus, and forms Choosing and using colors, fonts, and other visual elements to create a cohesive and visually appealing design Collaborating with team members, such as developers and project managers, to plan and implement web projects Creating wireframes, prototypes, and mockups to visualize and test design ideas Using design software such as Adobe Photoshop, Illustrator, or Sketch to create visual designs/ Testing and debugging design issues to ensure that the website or web application is functioning properly.

    Pricing: 1k - 2k pesos

  • Website Maintenance

    Web maintenance refers to the ongoing process of updating, improving, and maintaining a website or web application. It involves ensuring that the website or web application is functioning properly, is up-to-date with the latest information and features, and is in line with the needs of the users. Some common responsibilities of a web maintenance team may include: Updating and adding new content to the website or web application Fixing any issues or bugs that are discovered Optimizing the website or web application for performance and security Ensuring that the website or web application is compatible with different web browsers and devices Monitoring website traffic and analytics to identify areas for improvement Collaborating with team members, such as developers and designers, to plan and implement updates and improvements Maintaining backups of the website or web application to protect against data loss.

    Pricing: 500 - 1k pesos

  • Website Hosting

    Web hosting is the process of storing and managing the files and data for a website or web application on a server. When a website is hosted, it is made available to users on the internet by connecting the server to a network such as the World Wide Web.

    Pricing: 2k - 5k pesos
